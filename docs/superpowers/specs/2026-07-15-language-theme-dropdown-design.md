# 双语与主题下拉设计

## 目标

在不改变现有业务功能和页面结构的前提下，把顶部语言与外观控制改为两个紧凑下拉框，并让商业前端在中文和英文之间即时切换。彩色主题升级为明显区别于浅色主题的高饱和极光玻璃风格。

## 交互

- 顶部右侧顺序为“语言下拉、外观下拉、账户操作”。
- 语言下拉包含“中文、English”，默认中文；切换后立即更新当前界面，刷新后保持，并同步 `<html lang>`。
- 外观下拉包含“浅色、暗色、彩色”，保留现有主题持久化和跨标签页同步。
- 创作页左侧“灵感光谱”和右侧“灵感色板”继续复用同一外观组件，统一变为下拉。
- 下拉基于 Radix Select，支持键盘、Esc、外部点击关闭和移动端触控。
- 手机端顶部只显示紧凑图标/短标签，展开菜单仍显示完整选项，避免挤压登录注册按钮。

## 架构

新增轻量 `LanguageProvider`。语言资源使用类型安全的扁平字典，页面通过 `useLanguage()` 获取 `locale`、`setLocale` 和 `t()`。用户提示词、作品内容、用户名、邮箱和后端返回错误保持原文，其余商业界面固定文案按当前语言渲染。

`ThemeSelector` 从三按钮 radiogroup 改为受控 Select。主题值与原 localStorage key 不变，避免升级后丢失用户选择。

## 彩色主题

- 背景使用青蓝、钴蓝、紫罗兰、玫红和少量金橙叠加的高饱和极光网格。
- 顶栏和卡片保持半透明浅色玻璃，增加彩色边缘光和更清晰的层次。
- 激活导航、注册按钮和主要操作使用蓝紫玫红渐变。
- 创作区中心内容维持可读性，只增强环境光、面板描边和选择器高光，不把正文区域变成高饱和色块。
- 浅色、暗色主题保持现有方向。

## 翻译范围

覆盖顶部导航、首页、图像创作、生成预览、作品、灵魂画廊、积分、账户、登录和注册。灵魂画廊按语言切换中英文标题主次；完整提示词不翻译。

## 测试与验收

- 语言值规范化、中文/英文资源和动态插值。
- localStorage 读取写入与 `<html lang>` 同步。
- 语言、外观组件均渲染为下拉，不再出现三个平铺主题按钮。
- 中英文导航顺序不变。
- 中文默认页和英文页关键文案渲染正确。
- 1920×1080、1366×768、390×844 无横向溢出；键盘和触控可操作。
- `npm run lint`、`npm run test`、`npm run build`、`git diff --check` 全部通过。
