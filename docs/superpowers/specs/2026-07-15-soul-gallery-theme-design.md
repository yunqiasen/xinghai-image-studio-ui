# 灵魂画廊与主题视觉设计

日期：2026-07-15

## 目标

在不改变现有创作工作台功能和居中排版的前提下，增加独立的“灵魂画廊”；暗色主题调整为深蓝黑、青蓝雾光与浅紫玫紫渐变，彩色主题保持增强前的克制极光配色。

## 页面与信息架构

- 顶部导航顺序固定为：首页、创作、作品、灵魂画廊、积分、我的。
- 新页面路径为 `/soul-gallery`，它是静态精选模板库，不混入当前用户的“我的作品”。
- 页面保持现有 `max-width: 1500px` 外框和四周留白。桌面端内容最大宽度进一步收敛到 `1320px`，移动端单列。
- 首屏由标题、说明、搜索框、来源统计和分类筛选组成；下方使用图片主导的响应式瀑布流卡片。
- 卡片点击打开详情弹窗。弹窗左侧大图，右侧显示来源、分类、完整提示词、“复制提示词”和“带入创作”。

## 素材与数据

- 收录 19 项：imagic6 8 项、imya 11 项。
- 素材复制到 `public/soul-gallery-assets/imagic6` 与 `public/soul-gallery-assets/imya`，运行时不依赖外站。
- 数据集中在 `src/app/soul-gallery/catalog.ts`，每项包含 id、标题、英文标题、来源、分类、图片、宽高、描述和完整提示词。
- 搜索覆盖中英文标题、分类、来源、描述和提示词；分类使用纯前端过滤，不新增后端接口。

## 带入创作

“带入创作”通过 React Router 的 location state 传递提示词到 `/studio`。创作页只在收到非空提示词时替换当前输入框，并立即清除 history state，避免刷新或后续返回重复覆盖用户编辑。提示词长度上限调整到 4000，以容纳精选模板中的完整提示词；现有生成模式、模型、比例、张数、分辨率、上传和异步任务流程不变。

## 视觉

- 画廊风格：暗色展陈空间、克制玻璃面板、图片边缘柔光、紧凑瀑布流；视觉语言参考素材站，但不复制其页头、文案和布局比例。
- 暗色：底色 `#071022` 至 `#11152f`，左上青蓝雾光，右上浅紫玫紫光晕，面板为半透明深靛蓝。
- 彩色：沿用增强前的浅灰蓝底色与青、珊瑚、紫、金四色边缘光晕；中心阅读区、导航和主要按钮保持克制，不叠加高饱和渐变。
- 浅色：保持现状。
- 动效只用于卡片悬浮、弹窗和筛选状态，150–300ms，并遵守 `prefers-reduced-motion`。

## 可访问性与响应式

- 所有可点击卡片使用按钮语义，键盘可聚焦，弹窗支持 Esc 关闭和焦点管理。
- 图片提供明确 alt；图像保留宽高比并懒加载，避免布局跳动。
- 触控按钮最小高度 44px。
- 1920×1080、1366×768 使用 3–4 列瀑布流；390×844 使用单列，弹窗改为纵向滚动。

## 测试与验收

- 自动测试验证 19 项标题、提示词、图片路径、来源统计、搜索、分类和导航顺序。
- 自动测试验证路由提示词的读取与清理规则。
- 运行 `npm run lint`、`npm run test`、`npm run build` 与 `git diff --check`。
- 浏览器验证三种尺寸、暗色与彩色主题、搜索、筛选、详情弹窗、复制和带入创作。
